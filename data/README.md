## Data

DELPHI requires spatial transcriptomics datasets with paired H&E images.
Download the following datasets from their original sources:

| Dataset | Source |
|---------|--------|
| HER2ST | https://doi.org/10.1038/s41467-021-26271-2 |
| cSCC | https://www.ncbi.nlm.nih.gov/geo/query/acc.cgi?acc=GSE144239 |
| PRAD (HEST) | https://huggingface.co/datasets/MahmoodLab/hest |
| TCGA-BRCA | https://portal.gdc.cancer.gov/projects/TCGA-BRCA |

Pre-trained model and pre-computed results:
